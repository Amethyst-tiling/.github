# Amethyst — Portable Tiling Window Manager (Free)

## About Amethyst

Amethyst provides a complete tiling window management environment for users who want automatic window layout. Amethyst combines tiling layouts, keyboard shortcuts, multi‑display support, and floating windows. Amethyst enables users to automatically arrange windows in a grid without manual resizing. Amethyst runs on desktop platforms. Amethyst is trusted by developers, designers, writers, productivity enthusiasts, and anyone who wants efficient window management.

## Deep Dive — Tiling Window Manager Notes

Tiling layouts: vertical, horizontal, fullscreen, tall, wide, column, row, grid
Keyboard shortcuts: change layout, focus window, move windows, resize gaps
Multi‑display: each display has independent layout, move windows between screens
Floating windows: whitelist apps, make certain windows float above tiling
Gaps: adjustable gaps between windows, outer margins, inner padding
